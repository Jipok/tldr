# dvtm

> Dynamic virtual terminal manager.

- Show manual page:

`Ctrl+g ?`

- Create a new shell window:

`Ctrl+g c`

- Focus the **n**-th window:

`Ctrl+g **n**`

- Close focused window:

`Ctrl+g x x`
`Ctrl+d (for shell only)`

- Toggle between defined layouts:

`Ctrl+g Space`

- Toggle dvtm mouse grabbing:

`Ctrl+g M`

- Quit dvtm:

`Ctrl+g q q`
