# abduco

> Terminal session manager.

- List sessions:

`abduco`

- Attach to a session, create if not exists:

`abduco -A {{name}} bash`

- Attach to a session with dvtm, create if not exists:

`abduco -A {{name}}`

- Detach from session:

`Ctrl + \`

- Attach to a session in readonly mode:

`abduco -Ar {{name}}`
